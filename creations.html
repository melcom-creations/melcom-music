🔗 melcom's FFmpeg Audio Normalizer v3.0.0 (Halloween Edition)

melcom's FFmpeg Audio Normalizer is a free Windows GUI for automatic loudness normalization using the EBU R128 standard, powered by the versatile FFmpeg engine. It supports MP3, WAV, FLAC, AAC, OGG, and more, with batch processing and drag-and-drop convenience. Normalized files are saved alongside the originals with the suffix -Normalized.

I built this tool to make audio normalization simple and accessible — no command line required. Written in Python, it’s available in English, German, and Polish, each with a dedicated help file. Questions or issues? Contact me.

📌 Note: Requires FFmpeg. Download it from this page. An included help file explains how to set the ffmpeg.exe path in the program’s options.

What's new in v3.0.0 (Halloween Edition)?

  • Batch Processing: Process hundreds of files at once! You can now add multiple files or entire folders to the processing queue.
  • Integrated Audio Player: A new mini-player with playlist controls (Play, Stop, Next, Previous) lets you preview your audio files directly in the app.
  • Completely Redesigned UI: The interface has been modernized with a two-column layout, a batch progress bar, and a live status bar for a vastly improved workflow.
  • Live Input Validation: Custom LUFS and True Peak fields now instantly highlight invalid values in red to prevent errors before you start.
  • Output Format Info: Selecting an output format now displays helpful details about its technical specifications and typical use cases.

To convert the samples, you will need FLAC. You can create a batch file to use the Flac.exe, or you can use Flac Frontend. I can also provide you with my batch file. Please write to me if you would like to have it.

🔗 🖍️ Highlighter with X – Tampermonkey Script (v1.3.0)

Highlighter with X is a compact userscript that lets you highlight any text on any webpage in yellow, green, or red, and remove the highlight instantly with a hover-activated “×” button.

It’s perfect for marking up patch notes, articles, or forum posts. The script also automatically saves your highlights and restores them when you revisit the page.

  • Custom colors for green & red
  • Black text for maximum readability in dark mode
  • Hover-X with white border and larger click area
  • Keyboard shortcuts: ALT+1/ALT+2/ALT+3 to highlight, ALT+0 to remove
  • Auto-saves highlights and restores them later

⚙️ Requires the free Tampermonkey browser extension to run.
