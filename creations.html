It supports MP3, WAV, FLAC, AAC, OGG, and more, with batch processing and drag-and-drop convenience. Normalized files are saved alongside the originals with the suffix -Normalized.

I built this tool to make audio normalization simple and accessible ‚Äî no command line required. Written in Python, it‚Äôs available in English, German, and Polish, each with a dedicated help file. Questions or issues? Contact me.
