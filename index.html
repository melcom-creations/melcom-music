🚀 New Home on GitHub Pages!

A Faster, Modern Experience

I'm excited to announce that the website has officially moved from Blogger to a new, modern platform hosted on GitHub Pages.

This move was a big project, but it brings some fantastic benefits. The new site is built with pure HTML and CSS, making it significantly faster, cleaner, and more reliable. It also gives me complete control to make updates and add new features more easily in the future.

All the music and content you're used to are still here, just in a much speedier package. Thanks for your support during this transition. I hope you enjoy the new experience!

Please note: I'll be actively working on the site today and over the next few days to iron out any kinks. If you happen to notice any temporary issues, please just wait a few minutes and refresh. Thank you for your patience!

Cheers,
