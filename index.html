🚀 Update: melcom's FFmpeg Audio Normalizer v2.2.2 Released!

Featuring Audio Previews and a Real-Time Process Log

I'm happy to announce a significant update to my FFmpeg Audio Normalizer. Version 2.2.2 introduces two powerful features designed to improve your workflow and provide much better feedback during the normalization process.

  • New Audio Preview: You can now use the new "Play" and "Stop" buttons to instantly listen to your selected audio file before processing. This helps ensure you've loaded the correct track without leaving the application.
  • Live Process Information: The process window has been completely overhauled. It now shows a real-time log of FFmpeg's output, so you can see exactly what's happening at every step. A scrollbar has also been added for convenience.

🚀 New Home on GitHub Pages!

A Faster, Modern Experience

I'm excited to announce that the website has officially moved from Blogger to a new, modern platform hosted on GitHub Pages.
