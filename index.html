Professional Two-Pass Normalization and Quality Preservation

I'm excited to announce the release of FFmpeg Audio Normalizer v3.1.0 (Feierabend Edition)! üçª This version is a major milestone, introducing professional-grade features that make the tool even more versatile for musicians and audiophiles.

Key highlights of this update:

  • Linear Two-Pass Normalization: A high-precision mode that analyzes audio first to calculate a perfect gain adjustment while keeping the original dynamic range (LRA) intact.
