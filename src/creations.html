🔗 melcom's FFmpeg Audio Normalizer v4.1.0 (Kontrollraum Edition)

melcom's FFmpeg Audio Normalizer is a free Windows GUI for automatic loudness normalization using the EBU R128 standard, powered by the versatile FFmpeg engine. It supports MP3, WAV, FLAC, M4A, OGG, and more. Welcome to the Kontrollraum Edition - where raw audio signals meet absolute Kontrolle. Version 4.1.0 expands the Kontrollraum Edition with configuration profiles, a modularized architecture, an interactive graphic equalizer, and numerous workflow refinements.

I built this tool to make audio normalization simple and accessible - no command line required. Written in Python, it's available in English, German, Polish, and now Swedish, each with its dedicated help file. Questions or issues? Contact me.

📌 Note: Requires FFmpeg. Download it from this page. An included help file explains how to set the ffmpeg.exe path in the program's options.

What's new in v4.1.0 (Kontrollraum Edition)?

Version 4.1.0 is the biggest evolution of the project so far. What started as a simple loudness normalizer has grown into a complete audio management and inspection environment with professional metadata handling, advanced export controls and significantly improved day-to-day workflows.

  • Configuration Profiles for saving, loading and switching complete normalization setups through portable JSON files
  • Audio Properties Inspector featuring General, Details, Artwork and Statistics tabs with live background synchronization
  • Lossless Metadata & Artwork Management including viewing, editing, exporting, replacing and removing embedded cover art
  • Professional Audio Analysis with EBU R128 loudness, True Peak, RMS, Crest Factor, DC Offset and detailed signal statistics
  • Advanced Export Controls with selectable sample rates, bit depths, bitrates and format-specific quality settings
  • Interactive Audio Player with seeking, pause/resume support and real-time timeline updates
  • Native Drag & Drop support for files and folders with duplicate protection and background scanning
  • FFmpeg Diagnostic Engine translating technical error messages into understandable explanations
  • Explorer Integration with instant file-location access and native Windows properties support
  • Fully Modular Architecture prepared for future expansion and long-term maintainability
  • Extensive Localization with dedicated help files and support for English, German, Polish and Swedish
  • And yes... we finally have an Equalizer. 😊

The Kontrollraum Edition is focused on one goal: giving you complete Kontrolle over every stage of your audio normalization workflow.

For a full list of changes, see the included CHANGELOG.md.

🔗 🔧 melcom's GhostKey Sentinel v1.1

melcom's GhostKey Sentinel is a free, lightweight AutoHotkey script designed to fix mechanical keyboard chattering-the frustrating issue where a single key press registers multiple times.

This "set it and forget it" tool runs silently in the background, intercepting unwanted "ghost" key presses without adding any input lag. It's the perfect solution to restore a flawless typing experience on keyboards that have started to show their age.

📌 Note: Requires the free AutoHotkey v2 software to be installed on your system.

What's new in v1.1?

  • Bug Fix: Fixed a conflict with the Steam Overlay. The script now ignores the Tab key to ensure the Shift + Tab shortcut works reliably in all applications and games.

Key Features

  • Solves frustrating double-typing issues (key chattering).
  • Runs silently in the background with minimal resource usage.
  • Adds no noticeable input lag to your typing.
  • Easy "set it and forget it" installation.
  • Fully open-source and customizable.

🔗 🖍️ Highlighter with X - Tampermonkey Script (v1.3.0)

Highlighter with X is a compact userscript that lets you highlight any text on any webpage in yellow, green, or red, and remove the highlight instantly with a hover-activated "×" button.

It's perfect for marking up patch notes, articles, or forum posts. The script also automatically saves your highlights and restores them when you revisit the page.

  • Custom colors for green & red
  • Black text for maximum readability in dark mode
  • Hover-X with white border and larger click area
  • Keyboard shortcuts: ALT+1/ALT+2/ALT+3 to highlight, ALT+0 to remove
  • Auto-saves highlights and restores them later

⚙️ Requires the free Tampermonkey browser extension to run.
